## ⚡ SYSTEM OVERVIEW

Portfolio cá nhân được xây dựng với phong cách **Futuristic / Cyberpunk**, phản ánh hành trình chuyển mình từ **Kỹ sư CNTT (IT Engineer)** sang **Chuyên viên Phân tích Dữ liệu (Data Analyst)**.

Không chỉ là một trang giới thiệu thông thường — đây là một **trải nghiệm tương tác** được thiết kế để thể hiện tư duy kỹ thuật lai (Hybrid Engineering Mindset): hiểu hạ tầng hệ thống, thành thạo dữ liệu, và giao tiếp insight bằng ngôn ngữ kinh doanh.

## 📊 PROJECTS SHOWCASED

### 🗄️ Asset Management Intelligence System

> *IT Engineer × Data Analyst Hybrid Project*

- Phân tích toàn diện **vòng đời giao dịch thiết bị** trong môi trường doanh nghiệp
- Thiết kế **SQL Schema phức tạp** với ràng buộc toàn vẹn dữ liệu chặt chẽ
- Xây dựng **dashboard phân tích** giúp phát hiện điểm thất thoát tài sản theo thời gian thực
- Kết quả: **giảm thiểu đáng kể rủi ro mất mát tài sản** thông qua data-driven decision making

**Kỹ năng thể hiện:** SQL · Data Modeling · Business Intelligence · Asset Lifecycle Analysis

---

### ⚙️ Automated Data Pipeline (CI/CD)

> *DevOps meets Data Engineering*

- Xây dựng quy trình **ETL-style** tự động hóa việc triển khai mã nguồn
- Đảm bảo **tính toàn vẹn dữ liệu tuyệt đối (Zero Data-Loss)** xuyên suốt pipeline
- Tích hợp **kiểm tra tự động** và rollback mechanism khi phát hiện lỗi
- Minh chứng khả năng **tư duy hệ thống** — từ code đến infrastructure

**Kỹ năng thể hiện:** CI/CD · GitHub Actions · ETL Design · Data Integrity · Automation

      ▼

      │
      ▼
[Deploy → GitHub Pages]
      │
      ▼
